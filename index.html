What is Docker

Your answer here- Docker is a tool/ open source technology designed to create, deploy and run applications using container services.Container allows a developer to put together an application and all its required parts ( libraries and dependencies)and ship it out as one package;using Linux kernels. This eliminates the repetitive task of setting and configuring development environment.With the help of Docker developer can parallelly work on building a great software at a much faster rate and more efficiently.Source - www.docker.com, www.opensource.com

What is Nginx

Your answer here. - Nginx is a kind of software which is used in web servers to serve the concurrent requests ( can handle over 10,000 concurrent request). It is an open source software for web serving, reverse proxying, caching, load balancing, media streaming and more. NGINX can also function as a proxy for email and reverse proxy and laod balancer for HTTP, TCP and UDP servers. Source - www.nginx.xom, www.atulhost.com
